# Ocellaris WP - Custom Astra WP Theme

Tema custom para el sitio web de Ocellaris (ocellaris.com.mx). Se trata de un tema hijo de AstraWP, ajustado a la medida para las necesidades de este proyecto en concreto.

Autor: Daniel Limón  
dani@dlimon.net

## Guía rápida de uso

Para usar este tema, debes tener instalado el tema [Astra](https://wpastra.com) en tu instancia de WordPress.
Después, simplemente descarga este repo como un archivo `.zip` e instalalo como un nuevo tema en tu sitio web.
